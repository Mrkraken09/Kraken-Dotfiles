 # Install

 First we need to clone my repo using git
 ```
 $ git clone https://github.com/Mrkraken09/Kraken-Dotfiles.git
 $ cd Kraken-Dotfiles
 ```
 then we use stow to create symlinks

 ```
 $ stow {package name in dotfiles}
 ```
